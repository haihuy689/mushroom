# Mushroom

Mushroom is a fresh `Next.js` starter with a warm woodland-style landing page, built to be easy to publish on GitHub and deploy on Vercel.

## Stack

- `Next.js 16`
- `React 19`
- `TypeScript`
- CSS Modules + global CSS

## Local development

```bash
npm install
npm run dev
```

Open `http://localhost:3000`.

## Production check

```bash
npm run lint
npm run build
```

## Deploy

Push the repository to GitHub, then deploy with Vercel using either the dashboard import flow or the CLI:

```bash
vercel
```
